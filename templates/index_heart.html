AI Based Cardiac Health Check

ECG-Analyzer is a AI based app to classify digitized ECG readings to AAMI standards and also to identify if readings fall under the category of Myocardial Infraction.

TEST-NOW

Analyze Realtime/Batch Readings

Can handle single or multiple user data. Large number of user readings can be provided as a single input file. Results can be viewed in a tabular form for each user id.

Easy API Access

REST API for inference can be used, making integration with the digitizing app much easier. Helps to create a smooth integration between the ECG hardware to machine learning apps on the cloud or on premise

High Accuracy

Models trained on two different ECG datasets, with Myocardial Infraction detector using learnt representations via transfer learning.

Analyze ECG Samples

Use the Analyze-Now button to test the ECG sample you have.The input must be either csv,xls or xlsx extension, can contain one or multiple readings.

To take a look at a demo, use the DEMO button to view the data and the outputs.







DATA

Renowned and robust ECG datasets have been used to create the model.

PhysioNet MIT-BIH Arrhythmia: Consists of ECG recordings from 47 different subjects recorded at the sampling rate of 360Hz. Each beat is annotated by at least two cardiologists.The beat classifications are in accordance with Association for the Advancement of Medical Instrumentation (AAMI) EC57 standard.

PTB Diagnostic: This dataset consists of ECG records from 290 subjects: 148 diagnosed as MI , 52 healthy control, and the rest are diagnosed with 7 different disease. Each record contains ECG signals from 12 leads sampled at the frequency of 1000Hz. The model uses only ECG lead II.

App Features

Heartbeat Classification

To classify given ECG readings as one of the five different beat categories in accordance with Association for the Advancement of Medical Instrumentation (AAMI) EC57 standard.

Myocardial Infraction - Classifier

To classify if the ECG sample shows pattern similar to an incident of Myocardial Infraction. The model uses a combination of information available in the two datasets.
